# todo_crud_flask

A todo-list CRUD app built with Flask + MongoDB.


## Development

`python todo_crud_flask/run.py`


## Styles

`sass src/css/main.scss todo_crud_flask/static/css/style.css`


## Deploy

Push to heroku
